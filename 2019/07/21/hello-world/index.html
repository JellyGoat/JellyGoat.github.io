闲话

Thank you for watching this blog with Scient web tool to get to this web.
This is my first website for blog with GitHub Pages.

First 开始

这个博客的诞生是曲折的。一开始小金羊努力去写网页，后来改成使用Hexo，然后经历了三次Hexo爆炸之后，终于调教完毕，科学上网后发布了这些文章。
特别感谢Actinoi同志，为本博客的诞生做出了不可磨灭的贡献。
本博客于2019-07-21 14:16正式诞生。

Second 承平

我打算和Actinoi等各位大佬交换友链。
这个blog将承担我所有文章的发布。
其中有笔记、题解、闲话之类的，但是每一篇文章都是我仔细一点点写出来的。
希望大家能够有所收获qwq！

Third 结束

感谢Hexo和GitHub等平台的支持，若是可以，我将会把本站迁移到Gitee上访问，届时访问速度在国内会更快，而不是通过科学上网加速。
